# 🔐 Cryptography Toolkit

Java-based cryptographic toolkit implementing encoding, hashing, and symmetric encryption.

## Features
- Encoding/Decoding (Base64)
- Hashing (SHA-256)
- Symmetric Encryption (AES)

## Tech Stack
- Java 17+
- Maven

## How to Run
```bash
mvn clean package
java -cp target/classes CryptoOperations
